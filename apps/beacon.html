    # SensorFu Beacon

    Synchronize assets from SensorFu Beacon Home to Badrap.

    # Install App

    1. Open Beacon app page
    2. Click install
    3. Review the permission the app requests. Click Install the app
    4. Add home name and token you've received from SensorFu

    1. You're done
